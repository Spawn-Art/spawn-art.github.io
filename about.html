Spawn Art is a creative technology R&D lab committed to pushing the boundaries on using privacy-preserving tech to create imaginative, real-world web3 utility.

To achieve this we are creating a new open-source platform for developing Non-Fungible Programs (NFPs). NFPs are NFTs with encrypted executable components that run safely in browsers and native applications.

Personal programs

NFPs use private metadata in NFTs on Secret Network to ensure code is only accessible to the owner of the NFT.

Each NFP has unique source code, meaning customized and personalized experiences for each element in your collection.

Enhanced utility for NFTs

Encrypted code can be triggered by user interaction allowing for multiple layers of reveal in interactive content.

Applications

Interactive art Co-op gaming Personalized AI bots Private peer-to-peer communication Proof-of-authenticity

Endless flexibility

NFPs can incorporate any browser-compatible code, including JavaScript, WebAssembly, and Shaders.

Future development of API protocols will enable NFPs to function as plugins for a wide variety of third-party apps.

Spawn Art Genesis is the first NFP project developed by Spawn Art. It is an innovative NFT collection that showcases NFPs to allow owners to unlock rare traits and hidden features through community collaboration and problem-solving.

Owners of Genesis Spawn will also have priority access to new alpha from Spawn Art.
